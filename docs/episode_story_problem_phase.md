# 🎬 Edibuddy エピソード動画 - 課題編（冷蔵庫管理の困りごと）

## 📋 概要
冷蔵庫の中身把握と賞味期限管理の困難さ、そしてAI献立提案への期待を描く共感ストーリー

### ターゲット視聴者
- 30-40代の主婦・主夫層
- 冷蔵庫管理に悩む家族
- 食品ロスを減らしたいと思っている人

### トーン＆マナー
- 「あるある」感の高い日常描写
- テクノロジーへの期待感を醸成
- Edibuddyアプリの必要性を自然に感じさせる

---

## 🎭 メインキャラクター設定

### 主人公：田中 美咲（34歳）
- 職業：会社員（在宅ワーク多め）
- 家族構成：夫（36歳・会社員）、息子（8歳）、娘（5歳）
- 性格：効率重視、デジタル活用に前向き
- 悩み：冷蔵庫の中身把握、賞味期限管理、献立決めの負担
- 特徴：スマホ活用派、でも冷蔵庫管理は苦手

## 📖 冷蔵庫管理困りごと編（全6シーン）

### 朝の困惑から帰宅後の現実まで（6シーン）

#### シーン1：朝の冷蔵庫での困惑
- **時間**：AM 7:00
- **場所**：キッチン
- **描写**：冷蔵庫を開けて困った表情の美咲
- **心の声**：「トマトあったっけ？これ期限いつまで？」
- **行動**：奥を探す、期限チェック、においを嗅ぐ
- **問題**：食材の位置・状態・期限がわからない

#### シーン2：買い物前の記憶頼り
- **時間**：PM 2:00
- **場所**：キッチン
- **描写**：冷蔵庫を見ながらスマホにメモ
- **心の声**：「牛乳は...あるはず。卵は何個だっけ？」
- **不安**：記憶だけが頼り、確信が持てない

#### シーン3：スーパーでの迷い
- **時間**：PM 2:30
- **場所**：スーパー
- **描写**：牛乳の前で夫にメッセージ「牛乳ある？」
- **夫の返信**：「えーと...たぶんある」
- **心の声**：「結局わからない...買っておこう」

#### シーン4：帰宅後の重複発見
- **時間**：PM 4:00
- **場所**：キッチン
- **描写**：冷蔵庫を開けると牛乳が既に2本
- **心の声**：「やっちゃった...」
- **夫**：「昨日買ってきたんだった」
- **感情**：落胆とイライラ

#### シーン5：食材廃棄の発見
- **時間**：PM 4:10
- **場所**：キッチン
- **描写**：野菜室から萎びた野菜、期限切れ商品を発見
- **心の声**：「また800円分が無駄に...」
- **子供**：「もったいないね」
- **罪悪感**：環境と経済の両面で

#### シーン6：夜の無力感
- **時間**：PM 9:00
- **場所**：リビング
- **描写**：疲れた表情でため息
- **心の声**：「また同じ失敗...どうして管理できないんだろう」
- **願い**：「食材を無駄にしたくない」

## 🎨 課題の深刻さを伝える演出指針

### 視覚的な課題表現
- **見えない冷蔵庫の奥**：手を伸ばしても届かない食材、混沌とした内部
- **期限切れ商品のクローズアップ**：日付の赤字、変色した食材の質感
- **散乱したメモ**：紙のメモの不完全さ、情報の断片化
- **困惑の表情**：迷い、不安、罪悪感の細やかな感情表現

### 心理的課題の明確化
- **情報把握の困難**：何があるか、どこにあるか、いつまでか
- **記憶力の限界**：人間では管理しきれない複雑さ
- **家族間の情報共有不足**：コミュニケーションギャップ
- **無駄への罪悪感**：環境・経済・倫理的な重圧

### 音響での問題表現
- **ため息**：冷蔵庫を開けるたびの困惑
- **スマホのタップ音**：情報を探す必死な行動
- **ゴミ箱の音**：食材を捨てる重い決断
- **沈黙**：家族とのすれ違いの瞬間
